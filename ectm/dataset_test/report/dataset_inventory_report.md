## D. 评测集设计说明

本次评测采用**精准抽样策略**，以单案例端到端全链路覆盖为核心目标，而非追求文档数量规模：

- **难度分布**：101份文档覆盖标准件（约60%）、复杂表格件（约20%）、低质扫描件（约20%）三级难度谱系
- **全链路激活**：DEMO-MR-2026 案例完整激活 A1→A3、A5→A7 共6个Agent节点（A4因评测集暂无收益测算文档未激活）
- **标注质量**：双人交叉盲标，Cohen's κ = 0.918，分歧均为格式类差异，不影响实质内容
- **冲突注入**：5个精心设计的场景，覆盖参数矛盾、资质缺失、格式差异三类，并含真负例（CONF-003/004）验证
- **BBox可追溯**：MinerU evidence_map 覆盖率 100%，每个关键字段可精确追溯至原始文档页面坐标
