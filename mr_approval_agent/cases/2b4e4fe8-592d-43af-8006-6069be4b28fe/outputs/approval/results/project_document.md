# 医疗设备立项建议书

## 一、项目概述

本项目拟为医院放射科引进1台**医用磁共振成像系统（多核）**，设备场强为3.0T，具备多核代谢成像能力。该设备将用于支撑肿瘤、神经退行性疾病及心血管疾病的精准诊疗体系建设，填补当前无创、定量、早期评估代谢功能的影像技术空白。

根据申报材料《2026年度万元以上设备预算申报论证表3.0T-20260113.docx》，当前医院缺乏支持多核成像等高级功能的MRI设备，难以满足日益增长的精准医学需求。本次采购旨在通过引入具备前沿成像能力的高场强磁共振系统，提升医院在重大慢病领域的科研与临床服务能力。

## 二、临床需求与必要性分析

### （一）临床痛点明确

当前医院在肿瘤、神经系统疾病及心血管病诊疗中面临以下核心挑战：

- **缺乏无创代谢评估工具**：传统MRI主要提供解剖结构信息，无法对组织代谢状态进行定量分析；
- **精准诊疗能力受限**：如脑肿瘤边界界定、阿尔茨海默病早期诊断、心肌活力评估等场景亟需功能性成像支持；
- **科研转化瓶颈突出**：多核MRI（如¹³C、²³Na、³¹P）是国际前沿研究方向，但现有设备不支持相关序列开发与应用。

3. **扩展性强**：预留多核射频通道接口，未来可升级¹³C/²³Na等代谢成像模块；
4. **服务保障完善**：飞利浦在上海设有区域服务中心，响应时效优于部分国产品牌。
